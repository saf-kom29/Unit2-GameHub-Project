Concept #3: Transform üîÑ

üìê
Transform: Moves, scales, rotates, or skews elements WITHOUT affecting layout!
translateY(-2px) = Move UP 2 pixels
translateY(2px) = Move DOWN 2 pixels
scale(1.1) = Grow to 110% size
rotate(45deg) = Rotate 45 degrees

Transform vs Margin üéØ

‚ùå Using Margin
.btn:hover {
  margin-top: -2px;
}

‚Ä¢ Shifts other elements
‚Ä¢ Poor performance
‚Ä¢ Causes layout reflow
‚úì Using Transform
.btn:hover {
  transform: translateY(-2px);
}

‚Ä¢ Doesn't affect layout
‚Ä¢ GPU accelerated
‚Ä¢ Buttery smooth!
Always use transform for animations!

Button Lift Effect üöÄ

Multiple Transitions üé®

Understanding "ease" üìà

transition: transform 0.3s ease;
ease = Slow start, fast middle, slow end

Most natural feeling! (default)

linear = Constant speed throughout

Feels robotic, rarely used

ease-in-out = Very smooth curve

Extra polish for important elements

"ease" works great for 90% of cases!
