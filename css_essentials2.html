Media Query Not Working

The mobile styles aren't applying. What's wrong?

/* At the TOP of the CSS file */
@media (max-width: 768px) {
    .hero { padding: 60px 0; }
}

/* Later in the file */
.hero {
    padding: 120px 0;
}

‚úÖ Fix: Media queries should be at the END!

CSS cascades ‚Äî later rules override earlier ones.
The .hero rule comes AFTER the media query, so it always wins! 

/* Base styles FIRST */
.hero { padding: 120px 0; }

/* Media queries at END */
@media (max-width: 768px) {
    .hero { padding: 60px 0; }
}
