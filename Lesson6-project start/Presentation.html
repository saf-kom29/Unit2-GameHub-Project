Concept #1: CSS Reset üîÑ

Every professional site starts with a reset! 
/* Reset browser defaults */
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
Why? Browsers add default margins/padding. The * selector removes them ALL so you start with a clean slate.

Box-Sizing: Border-Box üì¶

‚ùå Without border-box
width: 200px;
padding: 20px;

Total = 240px!
(200 + 20 + 20)
‚úì With border-box
width: 200px;
padding: 20px;

Total = 200px!
(includes padding)
border-box makes width calculations EASY!

üéØ

‚Ä¢ max-width keeps content readable

‚Ä¢ padding: 0 20px adds breathing room

Container in Action üñºÔ∏è

‚Üê Full Browser Width ‚Üí

You'll use .container in EVERY section!
